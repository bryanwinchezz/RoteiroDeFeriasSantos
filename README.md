# 🏖️ Roteiro de Férias: Santos

**RoteiroDeFeriasSantos** é um guia turístico interativo da cidade de Santos — SP, apresentando os melhores pontos turísticos, praias, restaurantes e atrações para quem quer aproveitar as férias na Baixada Santista.

## 📋 Sobre o Projeto

O **Roteiro de Férias: Santos** é uma aplicação web em página única que serve como guia turístico completo para a cidade de Santos/SP. O projeto apresenta sugestões de roteiros, pontos turísticos, opções de lazer e informações úteis para moradores e visitantes que querem explorar tudo que Santos tem a oferecer.

## 🗺️ O que você encontra no guia

- **Praias** — as melhores praias da orla de Santos
- **Pontos turísticos** — Aquário, Museu do Café, Bolsa do Café e muito mais
- **Gastronomia** — restaurantes e culinária local
- **Entretenimento** — opções de lazer para toda a família
- **Roteiros sugeridos** — programações para diferentes tipos de visita (1 dia, fim de semana, semana inteira)

## 📍 Sobre Santos

Santos é uma cidade litorânea localizada no estado de São Paulo, famosa pelo maior porto da América Latina, pelo Aquário de Santos, pelo bondinho do Gonzaga e por suas praias urbanas. A cidade fica a cerca de 80 km de São Paulo pela Rodovia dos Imigrantes.
